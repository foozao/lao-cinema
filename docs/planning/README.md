# Planning Documentation

This folder contains documentation for features that are **planned but not yet implemented**.

These documents describe the design and implementation approach for future features. They serve as:
- **Design specs** for upcoming work
- **Reference material** for implementation
- **Decision records** for why certain approaches were chosen
